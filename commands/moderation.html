Moderation Command Description
ban Ban a user from the server
kick Kick a user from the server
lockdown Lock the current text channel so @everyone can not speak
mute Mute a single user from speaking in the server
nsfw mark the current channel as nsfw or remove it from nsfw tag
clear clear an amount of text in a channel
slowmode Set the channels slowmode rate
topic Set teh topic for the channel that it is used in
unban Unbans a user from the server
unlockdown Unlocks the channel so @everyone can speak in the channel
unmute Allows a muted user to speak again in the server.
